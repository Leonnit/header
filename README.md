# 🏠 Ambience Design — Site Vitrine

## 🔍 Présentation

Site vitrine **One Page** statique (HTML / CSS / JS) pour présenter les services de home staging d'Ambience Design. L'objectif : convertir les visiteurs en clients grâce à un design élégant et des réalisations inspirantes.

| Projet | Site vitrine Ambience Design |
|---|---|
| **Type** | One Page statique |
| **Stack** | HTML5 · CSS3 · JavaScript ES6+ |
| **Statut** | ✅ Production — Mars 2026 |

---

## 🛠 Stack & Design

- **Typographies** — Cormorant Garamond *(titres)* + Jost *(corps)*
- **Couleurs** — Crème `#f7f4ef` · Bleu `#2D3FE0` · Or `#C9A96E`
- **Responsive** — Desktop / Tablette `≤900px` / Mobile `≤600px`
- **Animations** — Scroll reveal avec `IntersectionObserver`, survols CSS
